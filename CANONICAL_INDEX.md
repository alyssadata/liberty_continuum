# Canonical Index

This index defines what is considered official in Liberty Continuum Frameworks.

## Canonical bundles
Only **tagged releases** are canonical bundles.

## Canonical files (once included in a tagged release)
- `VISION.md`
- `SCOPE.md`
- `DEFINITIONS/` files included in the release
- `PROTOCOLS/` files included in the release

## Rule
If it is not in a tagged release, it is draft.
